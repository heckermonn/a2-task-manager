About Task Manager

What is it?

This website is a application designed to assist users to organize, manage and track their tasks. Users are able to create, set, assign priorities and Status.

Key Features?

  • User Identification
  • Task Sorting
  • Add, Edit & Delete Tasks

Designed with:

  • Frontend: HTML5, CSS, JavaScript
  • Backend: PHP
  • Database: MySQL

How To Use?

Click Add Task to create a New Task.

Use the Filter options to organize tasks by priority, status or due dates.

Click Edit to modify the Tasks.

Click Mark Done when a task is Done.

Login/Register to create an account with saved Tasks.
